# MP予測モデル / MP Prediction Models

日次データから限界利益（MP: Margin Profit）を予測する機械学習モデル集です。

## 📚 ノートブック一覧

### 1. 当月末MP予測モデル（日次データ版）

**ファイル**: `mp_current_month_prediction_model.ipynb`

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/goto-a-toast/MP_daily_pred_from_stock-Public/blob/claude/predict-monthly-mp-NcKkJ/mp_current_month_prediction_model.ipynb)

#### 概要
日次の確定値、A、B、C、Dの値から**当月末の限界利益**を予測するモデルです。

#### 特徴
- **入力**: 日次データ（確定MP、A、B、C、D、月、日）
- **出力**: 当月末のMP予測値
- **手法**: Random Forest + 月別・日別転換率のハイブリッド

#### 使用シーン
- 月の途中で**当月末のMP**を予測したい場合
- 日次の進捗データを活用したリアルタイム予測

#### 主な機能
- 日ごとの予測精度推移の可視化
- 月別・日別の転換率分析
- 月内予測の進捗シミュレーション
- 確定値と追加予測値の内訳表示

### 2. 次月スタートMP予測モデル（日次の次月データ版）

**ファイル**: `mp_unified_forecast_model.ipynb`

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/goto-a-toast/MP_daily_pred_from_stock-Public/blob/claude/unify-mp-forecast-models-kfJTN/mp_unified_forecast_model.ipynb)

#### 概要
当月X日時点の次月データ（次月確定、次月A、次月B、次月C、次月D）から、
**当月末時点の 次月確定+次月A（= 次月スタートMP）** を予測するモデルです。

#### 予測例
```
当月 2月12日時点
  次月確定 + 次月A = 4,571万円  ← 今日の実績
          ↓ 予測
  2月末時点の予測値 = 5,200万円  ← 月末までに +629万円 積み上がる
```

#### 特徴
- **入力**: 日次の次月データ（次月確定MP、次月A、次月B、次月C、次月D、月、日）
- **出力**: 当月末時点の 次月確定+次月A の予測値・成長量
- **手法**: Random Forest（特徴量：KA、B、C、D、日付、月）

#### 使用シーン
- 月の途中で**翌月のスタートMP**（次月確定+A）の着地を知りたい場合
- 月末に向けてどのくらい積み上がるか見通しを立てたい場合

#### 主な機能
- 今日の実績 vs 月末予測値の表示
- 日次推移グラフ（実績線 vs 予測線 + 成長分の塗りつぶし）
- 各日の「残り成長予測量」棒グラフ
- 特徴量重要度の表示

## 📊 データフォーマット

### 必要なCSV形式

| 列 | 列名 | 説明 | 使用するモデル |
|----|------|------|---------------|
| 0 | 行ラベル | 日付（例：2020/4/1） | 全て |
| 1 | 合計 / MP | 当月MP | モデル1 |
| 2 | 合計 / 確定MP | 当月確定MP | モデル1 |
| 3 | 合計 / AMP | 当月A | モデル1 |
| 4 | 合計 / BMP | 当月B | モデル1 |
| 5 | 合計 / CMP | 当月C | モデル1 |
| 6 | 合計 / DMP | 当月D | モデル1 |
| 7 | 合計 / 次月MP | 次月MP | （参考用） |
| 8 | 合計 / 次月確定MP | 次月確定MP | モデル2 |
| 9 | 合計 / 次月AMP | 次月A | モデル2 |
| 10 | 合計 / 次月BMP | 次月B | モデル2 |
| 11 | 合計 / 次月CMP | 次月C | モデル2 |
| 12 | 合計 / 次月DMP | 次月D | モデル2 |

**サンプルデータ：**
```csv
行ラベル,合計 / MP,合計 / 確定MP,合計 / AMP,合計 / BMP,合計 / CMP,合計 / DMP,合計 / 次月MP,合計 / 次月確定MP,合計 / 次月AMP,合計 / 次月BMP,合計 / 次月CMP,合計 / 次月DMP
2020/4/1,52612321,3560426,49051895,9945801,6513693,24336375,0,0,15815626,1787889,2610055,58101088
```

**旧フォーマット（12列・7列）にも対応**: 自動検出により過去のデータも使用可能

### エンコーディング
- UTF-8-sig（推奨）
- Shift_JIS（自動検出対応）

## 💡 使い分けガイド

| 知りたいこと | 使うモデル |
|-------------|-----------|
| **当月末のMP**を予測したい | モデル1: 当月末MP予測 |
| **翌月のスタートMP**（次月確定+A）の着地を知りたい | モデル2: 次月スタートMP予測 |

### 具体的な使い分け例（4月15日時点）

| 知りたいこと | 使うモデル | 使うデータ |
|-------------|----------|-----------|
| **4月末のMP** | モデル1 | 4月15日の当月データ（確定、A、B、C、D） |
| **4月末時点での5月スタートMP** | モデル2 | 4月15日の次月データ（次月確定、次月A、次月B、次月C、次月D） |
